# Cplusplus_Abstract_Types
與具象型別的類別相對，稱為抽象型別的類別。

抽象型別跟使用者的實作細節完全隔離開來，比較接近介面的概念。

       Interface >> Abstract Class >>  Concret Class >> Instance Object
       
                     >> 越往右邊，內部表述實作更具體。
                     
                     >> 越往左邊，越是需要捨棄成員變數。
